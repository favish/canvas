## All spacing props are available.

| prop          | css value      | type               |
| ------------- | -------------- | ------------------ |
| position      | position       | string             |
| flex          | flex           | string             |
| flexDirection | flex-direction | string             |
| flexBasis     | flex-basis     | number \|string    |
| flexGrow      | flex-grow      | number             |
| flexWrap      | flex-wrap      | "wrap" \| "nowrap" |
